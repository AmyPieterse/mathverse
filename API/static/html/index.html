Routes description

HTTP Methods Path Description
GET / Takes you to homepage
GET /users Gives back an array of all users
GET /users/:id Gives back an object of the user with matching ID
POST /register Encrypts password, saves successfully in DB, and gives back a JWT
PATCH/PUT /users/:id Updates user successfully (including encrypting password if one is given)
POST /login Checks for users in DB by email and compares passwords correctly. JWT is sent back.
DELETE /user/:id Removes user from database
GET /items Gives back an array of all items
GET /item/:id Gives back an object of the item with a matching ID
POST /items Saves items in database
PUT /item/:id Updates an item by an authenticated user
DELETE /item/:id Removes the product from database by an authenticated user
GET /user/:id/carts Authorized user can now retrieve all items added
POST /user/:id/cart Authorized user can add an item to the cart
PATCH /user/:id/cart Authorized user can update an item in the cart.
DELETE /user/:id/cart Authorised user can delete all items from cart
DELETE /user/:id/cart/:id Authorized user can delete a specific item from the cart
